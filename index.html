Activar los 3 workflows
En n8n, entra a cada workflow y activa el toggle "Active" (arriba a la derecha).
Deben quedar en verde. Si pide asignar credencial, selecciona Chatwoot API Key.

Copiar URL del Webhook 2 → Chatwoot

Luego ve a Chatwoot → Ajustes → Integraciones → Webhooks → Nuevo webhook
Pega la URL y activa solo el evento: conversation_created

Desactivar asignación automática de Chatwoot
Chatwoot → Ajustes → Entradas → WhatsApp Destidnos → Colaboradores

Así n8n toma el control total de las asignaciones.

En Dokploy → JMFlow → Create Service → Static
